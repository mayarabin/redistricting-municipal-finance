# Redistricting and Municipal Finance

This repository contains the data, code, geographic workflows, documentation, and supporting materials for Maya Rabin's Amherst College honors thesis, *Economic Effects of Splitting Cities When Redistricting: Effects of Gerrymandering Beyond Politics*.

The project examines whether splitting a municipality across multiple congressional districts affects the amount of federal intergovernmental funding that municipality receives.
